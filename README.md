# Nu-Challenge - API
> Application that authorizes transactions

Dependencies: 

1. express v4.17.1
2. data-store v3.1.0

# Use
> You will need to have nodejs installed on your computer.


# Installation

```sh
// Will update dependencies automatically
npm install
```

# Run
```sh
// listen http://localhost:3000/
npm run start
```

# Tests
```sh
npm run test
```
